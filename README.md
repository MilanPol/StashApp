# Stash App

Stash App is a local-first household inventory management application built with Kotlin Multiplatform and Compose.

## Documentation

The foundational domain design and architecture have been documented in the `docs` folder:

- [Bounded Contexts](docs/bounded_contexts.md) - Overview of the 7 bounded contexts (Inventory, Shopping, Sharing, etc.) and their relationships.
- [Domain Model](docs/domain_model.md) - Entities, Value Objects, and Domain Events per context.
- [Ubiquitous Language](docs/glossary.md) - Glossary of domain terms used throughout the project.
- [C4 Architecture Model](docs/c4_model.md) - Context, Container, and Component diagrams, along with core design decisions (local-first, selective event sourcing).
